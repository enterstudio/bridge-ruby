get_channel(channel)

Returns a reference to the channel specified by `channel`.

get_service(svc)

Returns a reference to the service specified by `svc`.

initialize(options = {})

Usage: Bridge::initialize(opts). Expects to be called inside an EventMachine block.

join_channel(channel, handler, fun)

Join the channel specified by `channel`. Messages from this channel will be passed in to a handler specified by `handler`. The callback `fun` is to be called to confirm successful joining of the channel.

options()
publish_service(svc, fun)

Broadcasts the availability of certain functionality specified by a proc `fun` under the name of `svc`.

ready(fun)

Similar to $(document).ready of jQuery as well as now.ready: takes callbacks that it will call when the connection handshake has been completed.

send(dest, args)

Calls a remote function specified by `dest` with `args`.
